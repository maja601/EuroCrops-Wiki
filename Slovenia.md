## Attribute table and feature description 
| Original attribute table column name |                                       |
| ------------------------------------ |---------------------------------------|

| GERK_PID                             | GERK polygon identification number    |

| AREA                                 | GERK area (in sq m)                   |
| **RASTLINA**                         | crop type in slovenian language       |
